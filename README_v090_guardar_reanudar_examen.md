# BOARDING AGENT TOOLS v0.9.0

## Mejora
Guardar y reanudar avance de examen mediante archivo JSON.

## Archivos a reemplazar/subir
- app.js
- version.json
- service-worker.js

## Uso
1. Inicia Examen diario, Simulador completo o Examen secuencial.
2. Durante el examen o en la retroalimentación, presiona "💾 Descargar avance".
3. Se descargará un archivo JSON con el orden del examen, respuestas y pregunta pendiente.
4. En Guía de Estudio, usa "📂 Cargar avance JSON" para continuar desde cualquier dispositivo.

## Nota
Para que el avance cargue correctamente en otro dispositivo, el banco de preguntas debe estar actualizado en ambos dispositivos.
